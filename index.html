About

This AI Tic Tac Toe Game project was born out of a passion for game development and artificial intelligence. The idea was sparked during a casual brainstorming session where the desire to create a challenging and adaptive Tic Tac Toe game met the curiosity of exploring AI algorithms.


The journey started as a personal project but quickly evolved into a collaborative effort with fellow enthusiasts. The project timeline reflects the dedication and enthusiasm we poured into crafting an engaging gaming experience.


This project serves as a Portfolio Project for Holberton School.
